# SMC Project Pipeline

This site documents the cleaned, manuscript-associated workflow used for the SMC project.

## What is included

- `raw_scripts/` as an untouched provenance archive
- `pipeline/` as the cleaned active workflow
- `config/` example files with placeholder paths
- `docs/` method, input, output, and mapping notes

## What is excluded

- raw sequencing data
- patient-level clinical metadata
- unpublished sensitive tables
- optional scripts that are not part of the default manuscript pipeline
