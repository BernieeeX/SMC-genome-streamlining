# Output Files

## 01_assembly_qc

- MetaWRAP assembly output directory
- filtered contigs FASTA
- QUAST report folder
- CheckM result folders
- dRep dereplication output

## 02_species_typing

- sourmash comparison CSV files
- GTDB-Tk classification output
- MLST summary table
- KleTy output files

## 03_annotation

- Prokka annotations per genome
- GFF, GBK, FAA, and related files

## 04_phylogeny_pangenome

- Panaroo pangenome output directory
- MAFFT alignment FASTA
- optional dRep dereplication output

## 05_amr_virulence

- ABRicate TSV files
- merged AMR summary table
- VFDB summary table

## 06_plasmid_mge

- Platon result directories
- MGEFinder output
- SPAdes plasmid assemblies

## 07_large_deletion

- assembly-based MUMmer delta, SNP, and tiling outputs
- read-mapping validation BAM, VCF, GVCF, and WHAM outputs
- large-deletion manifest table

## 08_rnaseq

- trimmed FASTQ files
- sorted BAM files and indexes
- featureCounts summary
- DESeq2 result table and serialized object

## 09_figures

- PDF templates for QC, AMR, RNA-seq, and structural-variation summaries
